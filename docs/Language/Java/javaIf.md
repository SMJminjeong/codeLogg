##### ✅ {  } 블록을 쓰지 않는다면 ? 
:::info
if 조건 다음에 나오는 오직 한 줄의 코드만 실행

```java
if (a > 10) a += 5;
```
:::  
<br/>
만약 if 다음에 두줄의 실행이 온다면 첫번째 줄만 실행되고 두번째 줄은 if조건과 관계없이 실행된다.

##### ✅ `if-else` 인 경우
:::info
else 가 필요없다면 `else-if` 만 존재해도 괜찮다.
:::
<br/>

##### 그 외
:::info
if 문에는 무조건 true/false 값의 조건이 들어가야한다.
:::
